Grundfunktioner
- Riktig backend (Firebase eller Supabase)
- Flera kontakter i sidopanelen, var och en med egen chatthistorik
- Meddelandepersistens via localStorage (överlever sidomladdning)

UX
- Slash-kommandon — /clear, /status, /who
- Meddelandereaktioner — minimalt [+1] räknare vid hover
- Ljudeffekter — tangenttryck/pip vid skicka/ta emot (avstängningsbar)
- Notisbadge — antal olästa på kontakt i sidopanelen

Statussystem
- Användare kan sätta en egen statustext
- Status syns i sidopanelen under kontaktnamnet
- /status-kommando för att uppdatera den

Rum / Kanaler
- #general, #random-stil kanaler vid sidan av 1-till-1-chattar
